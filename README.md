# 🏟️ Earthspan Protocol
**The RWA Layer for Environmental Recovery**

Project Earthspan tokenizes physical environmental impact into **Impact Metres**, creating a high-yield, institutional-grade asset class on the Solana blockchain. Built for the **2026 Solana Frontier Hackathon**.

---

## 🏛️ Project Vision

* **Track:** Real World Assets (RWA)
* **Mission:** Tokenizing land restoration via high-fidelity on-chain data.
* **Mechanism:** Empowering a decentralized workforce of **Burwa Bosses** to verify and scale global recovery efforts.
* **Utility:** Providing transparent, liquid, and programmable environmental assets for institutional portfolios.

---

## 📅 Roadmap (Q2 2026)

### Phase 1: April 2026 - Architecture & Foundation 🏗️
* Deployment of the Earthspan Core Program (Anchor/Rust).
