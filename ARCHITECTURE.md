# $UPRA Protocol Architecture

## 1. Asset Layer (RWA)
* **Standard:** Solana Token Extensions (Token-2022).
* **Asset:** "Impact Metres" (Unique land parcels).
* **Verification:** Decentralized Oracles linking physical growth to on-chain metadata.

## 2. Financial Layer (Treasury)
* **Token:** $UPRA (Fungible Liquid Impact).
* **Mechanism:** Automated yield distribution via Transfer Hooks.
* **Liquidity:** Constant-product pools for institutional entry/exit.

## 3. Operations Layer (Burwa)
* **Identity:** Burwa Boss IDs (Soulbound tokens for local leaders).
* **Incentives:** 1% automated royalty on all $UPRA secondary volume.
